# Audio Assets

The sound effects in this directory are provided by **Kenney** (https://kenney.nl).
They are licensed under **CC0 1.0 Universal** (Public Domain), meaning they can be used freely in projects.

## Asset Packs Used:

- Kenney Interface Sounds
- Kenney Sci-Fi Sounds

## Files:

- `click.ogg`: Interface interaction
- `error.ogg`: Invalid action / warning
- `ship.ogg`: Positive confirmation / shipping
- `doom.ogg`: Game Over / Explosion
- `victory.ogg`: Winning the game
- `defeat.ogg`: Losing the game
- `crisis.ogg`: Random event trigger
